# Half Adder using Verilog on Nexys 4 DDR FPGA

## Description
This project implements a Half Adder using Verilog HDL and is tested on the Nexys 4 DDR FPGA board.

## Inputs
- A
- B

## Outputs
- Sum
- Carry

## Software Used
- Xilinx Vivado

## Hardware Used
- Nexys 4 DDR FPGA

## Files
- Source Code
- Testbench
- Constraints (.xdc)
- Simulation Results
- RTL Schematic
- FPGA Output Images
